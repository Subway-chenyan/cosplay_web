# Issue Tracker

Issues 托管在 GitHub Issues。

## 使用方式

- **创建 issue**: `gh issue create --title "标题" --body "内容"`
- **列出 issue**: `gh issue list`
- **查看 issue**: `gh issue view <编号>`
- **添加标签**: `gh issue edit <编号> --add-label "标签"`
- **关闭 issue**: `gh issue close <编号>`

## 标签使用

使用中文标签进行 triage：
- 需要评估
- 需要信息
- agent自动化
- 需要人工
- 不会修复
